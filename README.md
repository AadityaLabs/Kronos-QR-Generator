# Kronos-QR-Generator
Kronos QR Generator is a fast and efficient tool for creating custom QR codes from URLs, text, contact details, and more. Generate high-quality, scannable QR codes instantly, making information sharing simple, secure, and accessible across devices and platforms.

# Kronos QR Generator

Kronos QR Generator is a modern browser-based QR code studio designed for high-fidelity output and flexible content generation. It supports text/URL, Wi-Fi credentials, vCard profiles, email templates, and SMS actions, with live canvas rendering and custom styling presets.

## Features

- Generate QR codes for:
  - Text and URL links
  - Wi-Fi configuration
  - vCard contact cards
  - Email composition links
  - SMS and telephone messages
- Live canvas preview with animated build-up rendering
- Solid or gradient QR foreground colors
- Multiple module styles: classic, rounded, dots, liquid
- Custom finder-eye styles and central logo branding
- High-resolution PNG export
- Clipboard image copy support
- Web Share support for mobile devices
- Input helpers, paste/clear controls, and password visibility toggles
- Ambient animated background and premium UI polish

## Built With

- HTML
- Tailwind CSS (CDN)
- JavaScript
- `qrcode-generator` library
- `canvas-confetti` library
- Lucide icons

## Project Structure

- `index.html` — main UI and layout
- `style.css` — visual styling, glass panels, and background effects
- `app.js` — QR generation logic, UI controls, canvas animation, and export functions

## Usage

1. Open `index.html` in your browser.
2. Select the content type tab (Text, Wi-Fi, vCard, Email, SMS).
3. Fill in the desired fields and watch the QR preview update in real time.
4. Customize the color mode, module style, and center logo.
5. Download the QR code as a PNG or copy it to the clipboard.
6. Share the generated QR asset using the Web Share API on supported devices.

## Notes

- No build tools or installation are required.
- The app runs entirely client-side in the browser.
- For full clipboard image copy functionality, use a browser with secure clipboard permissions.
